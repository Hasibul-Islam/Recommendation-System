## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the required libraries.

```bash
pip install numpy 
pip install pandas
pip install sklearn
pip install scipy
pip install math
pip install random
pip install matplotlib
pip install nltk
```
